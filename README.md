# EasyLibrary

#Framework
Spring Boot + Angular.js + MySQL
# Usage
Open the project in IntelliJ IDEA. Run the server.    
`cd angularjs`  
`ng serve --open` run the front-end server.  
View the home page on `localhost:4200/home`.
